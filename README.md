# Healthcare SQL Analysis 🏥

## 📌 Project Overview
This project focuses on analyzing a healthcare dataset using SQL to extract meaningful insights.

## 📂 Dataset
- File: healthcare_dataset.csv
- Contains patient records including age, disease, billing, etc.

## 🛠 Tools Used
- PostgreSQL
- SQL
- Excel (for cleaning)

## 📊 Key Analysis Performed
- Total patients count
- Average billing amount
- Most common diseases
- Age group analysis

## 📸 Results
Screenshots are available in the screenshots folder.

## 🚀 How to Run
1. Import CSV into PostgreSQL
2. Run queries from `analysis.sql`
3. View results

## 💡 Key Insights
- Example: Most patients fall in 30–50 age group
- Example: Billing is highest for XYZ disease
